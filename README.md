				- 변수 타입, 함수의 매개변수, 
				반환값의 정확한 이해
프로젝트 중 이해 안되는 부분은 개인이 공부해서 정리 후 설명

	1) 6명이니까 프론트에 집중하려면 자바스크립트, 리액트 위주로 
	백엔드에 집중하고 싶으면 자바, 스프링, 스프링부트 위주로
	3대3 하면 될듯
